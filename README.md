# BattleShip
A small Battleship game written in C, playable on the terminal, and later via a graphical interface

### Warning about the GUI version
The version with a graphical interface is not working; it is under development and is only compatible with macOS.

### 💻 MacOS Edition
The battleship.app file is the macOS executable for the game.

### 🪟 Windows Edition
The Windows version is not up-to-date but is functional (it does not have all features).

### 🛠️ MacOS file structure
The files are sorted according to their purpose (the files needed for the game engine are in /engine, the console client files are in /cli, and the GUI files are in /gui).

### 🌐 Online Mode 
LAN mode only works on the macOS version; therefore, you need to know your local IP address or the IP address of the game host. To test with two games running on the same computer, use the localhost IP address: 127.0.0.1
